Privacy Policy

Who We Are

LDN/ai (londonai.network) is a London AI ecosystem intelligence platform built by Jie Xi. Contact: jxi5410@gmail.com.

What Data We Collect

Company data: All company, investor, people, and funding data displayed on LDN/ai is sourced from public records including Companies House filings, public funding announcements, company websites, and news sources. No private or proprietary data is collected.

User accounts (optional): If you sign up, we store your email address, display name, and any connections/stars you create. This is stored in Supabase (EU West region, AWS eu-west-2) and is not shared with third parties.

Analytics: We use Vercel Analytics for anonymous, aggregated page view counts. No cookies are set for analytics. No personal data is collected by analytics.

API usage: The public API does not require authentication and does not log individual requests. The sourcing API (beta) requires an API key and logs usage for rate limiting only.

What We Don't Collect

  • We do not use tracking cookies
  • We do not use Google Analytics or any pixel-based tracking
  • We do not collect IP addresses for analytics
  • We do not sell or share any user data with third parties
  • We do not serve advertisements

Data Storage

All data is stored in Supabase (hosted on AWS eu-west-2, London region). The frontend is served via Vercel's global CDN. Source code is open source on GitHub.

Your Rights

You can request deletion of your account and all associated data by emailing jxi5410@gmail.com. If you believe any company or person data displayed on LDN/ai is inaccurate, you can request corrections via the same email.

Third-Party Services

  • Supabase — database and authentication (privacy policy)
  • Vercel — hosting and analytics (privacy policy)
  • Companies House — public company data (data policy)
  • Anthropic — Claude API for AI features (privacy policy)

Changes

We may update this policy. Changes will be posted on this page with an updated date.
