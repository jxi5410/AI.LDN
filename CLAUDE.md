# AI.LDN — Project Instructions

> Working style, preferences, and execution rules are in `~/.claude/CLAUDE.md` (global). This file contains project-specific context only.

## What This Is

London AI Ecosystem Intelligence platform. Interactive D3 force graph mapping 89 companies, 128 connections, 100 funding rounds ($37B+), 67 key people — with agent-ready public API and automated weekly data collection.

**Live:** https://londonai.network
**API:** https://ripugedrbnmvbbdntxgt.supabase.co/functions/v1/api

## Stack

- **Frontend:** Vite + React 18 SPA, D3.js (force graph), Recharts, vanilla CSS
- **Backend:** Supabase (Postgres, Edge Functions, pgvector, pg_cron)
- **AI:** Claude API (podcast summaries), Gemini API (embeddings 768d)
- **Hosting:** Vercel with Analytics
- **Languages:** HTML 81.2% (static pages in `/public`), JavaScript 18.8% (React SPA in `/src`)

```
                                                         → External APIs (CH, GitHub, Gemini, Claude)
```

Panels: Map · People · Events · News · Bits
Toggle: Company View / Investor View on the D3 graph

## Hard Constraints

- **Zero new npm dependencies.** Vanilla React + CSS + D3 only. No Tailwind, Framer Motion, shadcn.
- **Don't break URLs.** `/company/*`, `/insights/*`, `/ecosystem/*`, `/privacy`, `/terms`, `/sourcing` must all continue working.
- **Don't touch the D3 force simulation.** Improve the chrome, not the physics engine.
- **Don't touch `src/data.js`.** Data layer is clean and off-limits.
- **No box-shadow.** Use border + background changes for elevation.
- **No gradients, noise textures, parallax.** This is a data product.
- **Mobile breakpoint: 768px.** Below = mobile layout with hamburger + bottom sheet.

## Design System

Typography: DM Serif Display (headings) + DM Sans (body/UI) + JetBrains Mono (data/numbers)
Palette: Warm cream base (#faf9f5), terracotta accent (#C15F3C), 3-level elevation, 4-level text contrast
Identity: "Bloomberg Terminal meets Monocle magazine" — serious data, beautiful presentation
